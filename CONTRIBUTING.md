# コントリビューションガイド

このガイドへのフィードバック・改善提案を歓迎します。

## フィードバックの方法

### Issue

以下のような場合は Issue を作成してください：

- 誤字・脱字の報告
- リンク切れの報告
- 内容の誤りや不明点の指摘
- 新しいトピックの提案
- 改善のアイデア

### Pull Request

以下のような変更は Pull Request を歓迎します：

- 誤字・脱字の修正
- リンク切れの修正
- 説明の明確化
- 例の追加・改善

### 大きな変更について

構成の大幅な変更や新しいセクションの追加を検討している場合は、
まず Issue で提案・議論してから Pull Request を作成してください。

## コントリビューションの原則

このガイドの哲学に沿った貢献をお願いします：

1. **シンプルさを保つ** — 複雑さを増やさない
2. **検証可能であること** — 主張には根拠を示す
3. **実践的であること** — 理論より実践を重視する
